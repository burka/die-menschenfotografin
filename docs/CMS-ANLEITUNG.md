# CMS-Anleitung für Kathrin

## Einloggen

Öffne im Browser: **https://die-menschenfotografin.de/admin**
Melde dich mit deiner E-Mail-Adresse und deinem Passwort an.

---

## Bilder hochladen

### Schritt 1: Media (Rohdateien)
1. Klicke links auf **Media**
2. Klicke oben rechts auf **Create New**
3. Bild per Drag & Drop oder Klick hochladen
4. **Alt-Text** eingeben (kurze Beschreibung, z.B. "Brautpaar im Garten") – wichtig für Google & Barrierefreiheit
5. Speichern

### Schritt 2: Images (Bild mit Details)
1. Klicke links auf **Images**
2. **Create New**
3. Unter **Image** dein eben hochgeladenes Media-Bild auswählen
4. **Alt-Text** ausfüllen
5. Optional: **Caption** (Bildunterschrift), **Date** (Aufnahmedatum), **Story** (Geschichte zum Bild)
6. **Category** auswählen, damit das Bild einer Kategorie zugeordnet ist
7. Speichern

> **Tipp:** Lade Bilder vorher auf eine vernünftige Größe herunter (max. 2500px Breite reicht). Das spart Speicher und die Seite lädt schneller.

## Kategorien bearbeiten

Klicke links auf **Categories**. Dort siehst du deine Kategorien (z.B. Business & Event, Hochzeiten, Familie, Kindergarten).

### Kategorie-Einstellungen

| Feld | Was es macht |
|------|-------------|
| **Title** | Name der Kategorie |
| **Slug** | URL-Teil (z.B. `hochzeiten`) – **nicht ändern**, sonst funktionieren Links nicht mehr |
| **Sort Order** | Reihenfolge auf der Startseite (1 = ganz links, 4 = ganz rechts) |
| **Hero Image** | Großes Bild oben auf der Kategorie-Seite |
| **Preview Image** | Vorschaubild für die Kachel auf der Startseite |
| **Description** | Kurze Beschreibung der Kategorie |

### Inhalts-Blöcke (Content)

Das Herzstück! Hier baust du die Inhalte deiner Kategorie-Seiten zusammen. Klicke auf **Add Block** und wähle einen Block-Typ:

#### Galerie-Block
- Wähle mehrere Bilder aus der Images-Sammlung aus
- Werden als schönes Bilder-Raster angezeigt
- **Ideal für:** Foto-Galerien, Portfolio-Übersichten

#### Text-Block (Rich Text)
- Freitext mit Formatierung (fett, kursiv, Listen, Links)
- **Ideal für:** Beschreibungen, Konzepte, Abläufe

#### Bild + Text Block
- Ein Bild neben einem Text
- Du kannst wählen: **Bild links** oder **Bild rechts**
- Optionale Überschrift möglich
- **Ideal für:** "Über mich"-Abschnitte, Ablauf-Erklärungen

#### Überschrift-Block
- Eine große Überschrift mit optionalem Beschreibungstext
- Größe wählbar: H2, H3 oder H4
- **Ideal für:** Abschnitts-Trenner, Kapitel-Überschriften

> **Wichtig:** Du kannst die Blöcke per Drag & Drop in die gewünschte Reihenfolge ziehen! Einfach am Anfasser links greifen und verschieben.

## Website-Einstellungen

Klicke links unten auf **Site Settings** (unter "Globals").

| Feld | Was es macht |
|------|-------------|
| **Photographer Name** | Dein Name ("Kathrin Krause") |
| **Brand Name** | Markenname ("die Menschenfotografin") |
| **Tagline** | Dein Slogan ("Fine portraits for fine people") |
| **Contact → Email** | Deine Kontakt-E-Mail |
| **Contact → Phone** | Deine Telefonnummer |
| **Contact → Address** | Deine Adresse |
| **Impressum** | Rechtstext Impressum |
| **Datenschutz** | Rechtstext Datenschutzerklärung |

---

## Checkliste: Worauf achten?

- **Immer Alt-Texte** bei Bildern ausfüllen (gut für Google-Suche!)
- **Slugs nicht ändern** – sonst gehen bestehende Links kaputt
- **Bilder nicht zu groß** hochladen (max. ~2500px Breite, JPG-Qualität 80-85%)
- **Speichern nicht vergessen** – oben rechts auf "Save" klicken
- **Vorschau prüfen** – nach dem Speichern die Seite im Browser aufrufen und kontrollieren
- **Sort Order** bei Kategorien: kleinere Zahl = weiter vorne auf der Startseite

## Kurzübersicht: Was wo bearbeiten?

| Ich möchte... | Wo im CMS? |
|--------------|-----------|
| Neue Fotos hochladen | Media → Create New |
| Fotos mit Details anlegen | Images → Create New |

| Startseiten-Reihenfolge ändern | Categories → Sort Order anpassen |
| Kachelbild der Startseite ändern | Categories → Preview Image |
| Meinen Namen/Slogan ändern | Site Settings |
| Impressum aktualisieren | Site Settings → Impressum |
| Datenschutz aktualisieren | Site Settings → Datenschutz |
